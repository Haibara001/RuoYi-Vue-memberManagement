协会简介

城市足球协会成立于2010年，是经体育局批准注册的官方足球组织。我们致力于推广足球运动，组织各类活动，让更多人爱上足球。协会拥有专业教练团队、标准足球场地和完整的青训体系，每年服务超过5000名足球爱好者。我们的使命是培养足球人才，传播足球文化，促进全民健身运动发展。
